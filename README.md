# BioWaste — Driver Mobile

Case study redesign of a bio-waste route management tool, reframed as a consumer-grade native mobile app for truck drivers.

- Next.js 15 · React 19 · Tailwind 4
- Geist (Vercel) typography
- Phosphor Icons
- `html-to-image` PNG export at 1400×900

## Pages

- `/` — Case study showcase with all seven screens in a horizontal strip + component system.
- `/mockups` — Ten downloadable presentation mockups (1400×900 PNG, 2× pixel ratio).

## Local

```bash
npm install
npm run dev
```

Dev server runs at [http://localhost:3000](http://localhost:3000).

## Deploy

Optimised for Vercel. Any Next.js-compatible host works.
